O que são Unidades Monitoráveis (U.M.)?

Uma Unidade Monitorável (U.M.) é qualquer unidade de monitoramento na infraestrutura que precisa ser acompanhada para garantir performance, disponibilidade e cumprimento de SLA.

Importante: Este catálogo contempla os serviços inclusos nos tiers Essential e Pro. Para itens fora do catálogo, consulte nossos especialistas.

Plataforma de Monitoramento: A monitoração será realizada através da plataforma Zabbix, proporcionando visibilidade completa da infraestrutura.

Características Principais

Identificável
Possui uma identidade única dentro do ambiente AWS
Configurável
Permite aplicação de configurações específicas de monitoramento
Gerador de Observabilidade
Emite métricas, logs e eventos
Unidade de Cobrança
Serve como base para precificação do serviço de monitoramento
